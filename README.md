# Python学生就业管理系统 python950

## 项目概述

本项目是基于Python的学生就业管理系统，集成了学院、专业、企业、岗位、管理员以及学生信息的全面管理功能。系统旨在实现对学生就业相关信息的有效整理和跟踪。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Django
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0及以上版本

## 功能模块

- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理
  - 学生教育经历
  - 学生项目经历
  - 学生投递记录
- 修改个人信息与密码

## 系统角色

- 学生
- 管理员

## 目录结构


## 部署步骤

1. 安装Python 3.0及以上版本
2. 安装MySQL 8.0及以上版本
3. 安装项目所需的Python库（根据`requirements.txt`）
4. 配置数据库连接信息
5. 执行数据库迁移命令以创建数据库表
6. 运行开发服务器
